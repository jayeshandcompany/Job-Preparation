LLM -> Large Languag Model trained on large text datasets to understand and generate human-like text. 
    Examples include GPT-3, Gemini, and Claude  . 
    They are used in various applications such as chatbots, content generation, and language translation.   
    based on transformer architecture
Transformer architecture -> neural network architecture uses self-attention mechanisms to process sequential data.
    components include:
        Tokenization -> process of converting text into smaller units (tokens) for model processing.
        Positional Encoding -> converted into vectore representations that capture the position of each token in the sequence.
        self attention -> determine what has more weight and influence others
        Multi head attention -> Mulitple heads to focus on different parts of the input sequence simultaneously.
        Feed forward neural network -> processes the output of the attention mechanism to generate the final output.
    Benefits include:
        - Ability to capture long-range dependencies in text.
        - Parallel processing of input data, leading to faster training times.
        - Improved performance on various natural language processing tasks compared to previous architectures.
